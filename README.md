⏰ מערכת ניהול ובקרת שעות - מתן vs פיל
מערכת ווב (Web) עצמאית לניהול, מעקב והשוואת שעות עבודה. המערכת נבנתה כדי לאפשר שליטה מלאה על דיווחי השעות, זיהוי פערים מול דוחות המעסיק ("פיל"), וחישוב שכר מדויק כולל נסיעות.

✨ פיצ'רים עיקריים
⚡️ עבודה ללא התקנה: קובץ HTML יחיד שרץ בדפדפן. לא דורש שרת או אינטרנט (למעט טעינת ספריות).

📂 טעינת דוחות מעסיק: מנגנון חכם לטעינת קבצי Excel של המעסיק, עם זיהוי אוטומטי של עמודות (תאריך, כניסה, יציאה).

🤖 מילוי אוטומטי: אפשרות למילוי מהיר של שעות תקן (ראשון: 12:45-16:30, שאר הימים: 13:30-16:30).

📊 השוואה חכמה: זיהוי פערים אוטומטי בין דיווחי המעסיק לדיווחי העובד. פערים מסומנים בצבע (ירוק/אדום).

💰 מחשבון שכר: חישוב בזמן אמת של השכר המשוער לפי תעריף שעתי + החזרי נסיעות יומיים.

🚗 ניהול נסיעות: שליטה פרטנית על ימי נסיעות (Checkboxes) לחישוב מדויק.

📝 דוחות והדפסה: הפקת דוח PDF מקצועי ב-2 עמודים (טבלה + מכתב סיכום), מותאם להדפסה לרוחב (Landscape).

💾 שמירה וגיבוי: הנתונים נשמרים אוטומטית בדפדפן (Local Storage) לפי חודשים, עם אפשרות לייצא/לייבא קובץ גיבוי למחשב.

📱 שיתוף מהיר: כפתור לשליחת סיכום בוואטסאפ.

🚀 הוראות שימוש
1. התחלה
פשוט לחץ פעמיים על הקובץ index.html. המערכת תיפתח בדפדפן ברירת המחדל שלך (מומלץ Chrome או Edge).

2. זרימת עבודה חודשית (Workflow)
בחירת תאריך: בחר את החודש והשנה הרצויים בסרגל העליון.

טעינת נתוני מעסיק: לחץ על "📂 טען פיל" ובחר את קובץ האקסל שקיבלת.

הזנת נתונים אישיים:

מלא ידנית את שעות הכניסה/יציאה בעמודות הכחולות ("מתן").

או השתמש בכפתור "🤖 מלא אוטומטי" למילוי מהיר.

בדיקה: עבור על עמודת "הפרש". שורות עם פערים יסומנו באדום.

אם הכל תקין, סמן ✔ בעמודת "מאושר".

אם יש הערה, כתוב אותה בעמודת "הערות".

נסיעות: וודא שימים בהם הגעת לעבודה מסומנים ב-V בעמודת "נסיעה".

3. הפקת דוח סוף חודש
גלול למטה לאזור "עריכת מכתב נלווה" וודא שהטקסט מתאים לך.

לחץ על "🖨️ הדפס / PDF".

בחלון ההדפסה שנפתח, הקפד על ההגדרות הבאות:

Layout: Landscape (לרוחב).

Margins: Default (ברירת מחדל) או Minimum.

Destination: Save as PDF.

🛠️ הגדרות מתקדמות (שינוי שכר)
המחירים מוגדרים בתוך הקוד וניתנים לשינוי בקלות אם תקבל העלאה (🙏).

פתח את הקובץ index.html באמצעות "פנקס רשימות" (Notepad) או כל עורך טקסט.

גלול לתחתית הקובץ לאזור ה-<script>.

חפש את השורות הראשונות:

JavaScript

const RATE = 70;   // התעריף השעתי
const TRAVEL = 16; // תעריף נסיעות יומי
שנה את המספרים לערך החדש ושמור את הקובץ (Ctrl+S).

רענן את הדפדפן (F5).

📦 גיבוי ושחזור
המערכת שומרת נתונים בזיכרון הדפדפן. אם מחקת היסטוריה, הנתונים יעלמו. מומלץ: בסוף כל חודש, לחץ על "💾 גיבוי" ושמור את הקובץ במחשב שלך. כדי לשחזר, לחץ על "📥 טען גיבוי" ובחר את הקובץ ששמרת.

👨‍💻 טכנולוגיות
HTML5 / CSS3: עיצוב רספונסיבי ונקי.

Vanilla JavaScript: ללא פריימוורקים כבדים, לביצועים מהירים.

SheetJS (xlsx): ספרייה לקריאת קבצי Excel.

Local Storage: לשמירת נתונים מקומית.

Print CSS: התאמות ייחודיות להדפסה מושלמת.
